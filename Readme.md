<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Ability to run "go".

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jhilde7/GoFibonacciAPI.git
   ```
2. use Makefile to build under bin/Test
   ```sh
   Makefile build
   ```
3. use Makefile to run
   ```sh
   Makefile run
   ```
